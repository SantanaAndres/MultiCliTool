# 🛠️ MultiCLI Tool

[![.NET Version](https://img.shields.io/badge/.NET-10.0-blue)](https://dotnet.microsoft.com/)
[![C#](https://img.shields.io/badge/C%23-13.0-purple)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**MultiCLI Tool** es una herramienta de línea de comandos interactiva construida en C# que te permite realizar múltiples tareas de utilidad desde una misma interfaz: búsqueda de archivos, generación de contraseñas seguras y consulta del precio de Bitcoin (BTC) en tiempo real.

> ✨ Diseñada con una arquitectura por capas, inyección de dependencias y la hermosa biblioteca [Spectre.Console](https://spectreconsole.net/) para una experiencia visual atractiva en la terminal.

---

## 🚀 Características Actuales

| Módulo | Funcionalidad | Estado |
|--------|---------------|--------|
| 🔍 **Búsqueda de archivos** | Busca archivos por nombre en todo el sistema | ✅ Implementado |
| 🔐 **Generador de contraseñas** | Genera múltiples contraseñas seguras con longitud personalizable | ✅ Implementado |
| ₿ **Precio de Bitcoin** | Consulta el precio actual de BTC en USD desde CoinMarketCap API | ✅ Implementado |
| 🎨 **Interfaz interactiva** | Menús coloridos y prompts amigables con Spectre.Console | ✅ Implementado |
